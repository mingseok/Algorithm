## 두 실수의 곱

변수 a, b 에 각각 5.26, 8.27 을 넣어주고, 두 수의 곱을 소수점 셋째 자리까지만 출력하는 프로그램을 작성하세요.

<br/>

## 예상 출력:

43.500

<br/>

## 코드

```java
public class Main {
    public static void main(String[] args) {
        // Your Program Goes Here

        double a = 5.26;
        double b = 8.27;
        double c = a * b;

        System.out.printf("%.3f",c);
    }
}
```
