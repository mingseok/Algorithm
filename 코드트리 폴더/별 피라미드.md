## 별 피라미드

정수 n의 값을 입력받아 별표를 출력하는 프로그램을 아래 예를 참고하여 작성해보세요.

<br/>

## 입력:

4

<br/>

## 출력:

          * 
        * * * 
      * * * * * 
    * * * * * * * 


<br/>

## 코드

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        int a = sc.nextInt();

        for(int i = 0; i < a; i++) {

            for(int k = 1; k < a - i; k++) {
                System.out.print("  ");
            }

            for(int j = 1; j <= i * 2 + 1; j++) {
                System.out.print("* ");
            }
            System.out.println();

        }

    }
}
```
