## 날짜 변경하여 출력

mm-dd-yyyy 모양의 날짜 정보를 입력받아 yyyy.mm.dd 형식으로 출력하는 프로그램을 작성해보세요.

<br/>

## 입력

5-26-1994

<br/>

## 출력

1994.5.26

<br/>


## 코드

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        // Your Program Goes Here

        Scanner sc = new Scanner(System.in);

        String s = sc.next();

        String[] str = s.split("-");

        System.out.print(str[2] + "." + str[0] + "." + str[1]);

    }
}
```
